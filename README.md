# Dashboard Financiero ADATEC - Cartera

Sistema de análisis financiero de cartera con dashboard interactivo y chatbot con IA.

## Características

- 📊 Dashboard interactivo con visualizaciones de cartera
- 🤖 Chatbot con IA para análisis financiero (RAG)
- 🔐 Autenticación para acceso al chatbot
- 📈 Análisis de tendencias, eficiencia y riesgos
- 🚀 Desplegado en Vercel

## Documentación del Chatbot

Ver [CHATBOT.md](CHATBOT.md) para documentación completa del sistema de chatbot.

## Inicio Rápido

```bash
npm install
npm run dev
```

Acceder a:
- Dashboard: http://localhost:3000
- Chatbot: http://localhost:3000/chatbot.html

## Credenciales

Las credenciales de acceso se configuran en las variables de entorno (`DASHBOARD_USER` y `DASHBOARD_PASSWORD`).
Ver `.env.example` para el formato.
